⚙️ Google Drive & Gmail 設定

1Google Cloud Console でプロジェクトを作成
2「APIとサービス」→「ライブラリ」で Google Drive API と Gmail API を有効化
3「認証情報」→「OAuth 2.0 クライアントID」を作成（種類：ウェブアプリ）
4「承認済みのJavaScript生成元」に を追加
